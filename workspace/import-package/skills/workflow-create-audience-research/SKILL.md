# Workflow: Criar Pesquisa de Audiência

## Objetivo

Produzir um documento de pesquisa útil para estratégia, copy e conteúdo.

## Dono principal

`Market Researcher`

## Entradas

- `brief.md`
- materiais do cliente
- perguntas prioritárias do `Strategy Director`

## Passos

1. Ler o brief e listar fatos confirmados.
2. Mapear dores, desejos, objeções e linguagem do público.
3. Registrar concorrentes, alternativas e sinais de confiança.
4. Separar o que é evidência e o que é hipótese.
5. Concluir com implicações para estratégia e copy.

## Saída recomendada

- `clients/{client-name}/research/audience-research.md`

## QA mínimo

- público está claro
- objeções são acionáveis
- hipóteses estão marcadas
- existe bloco de implicações práticas
