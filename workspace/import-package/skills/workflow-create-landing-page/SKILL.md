# Workflow: Criar Landing Page

## Objetivo

Estruturar uma landing page coerente com campanha, oferta e canal de conversão.

## Donos principais

- `Landing Page Builder`
- `Copywriter`

## Entradas

- posicionamento
- campanha
- prova disponível
- CTA principal

## Passos

1. Definir objetivo da página.
2. Organizar a ordem das seções.
3. Redigir headline, prova, oferta, objeções e CTA.
4. Identificar fricções e pontos de confiança.
5. Registrar versão pronta para QA.

## Saída recomendada

- `clients/{client-name}/landing-pages/landing-page-v1.md`

## QA mínimo

- mensagem bate com anúncio
- CTA é claro
- objeções críticas foram tratadas
