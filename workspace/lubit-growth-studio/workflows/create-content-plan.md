# Workflow: Criar Plano de Conteúdo

## Objetivo

Traduzir a estratégia em uma rotina editorial executável.

## Donos principais

- `Content Strategist`
- `Copywriter`

## Entradas

- posicionamento
- pesquisa
- capacidade de produção
- canais do cliente

## Passos

1. Definir papel de cada canal.
2. Escolher linhas editoriais.
3. Criar calendário com frequência realista.
4. Relacionar cada pauta a um objetivo.
5. Sugerir reaproveitamento de formatos.

## Saída recomendada

- `clients/{client-name}/content-plan.md`

## QA mínimo

- calendário cabe na operação
- pautas têm objetivo
- existe conexão com conversão ou relacionamento
